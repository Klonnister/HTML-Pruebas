Center align elements

You can use any of these options to center elements horizontally:

  • margin: auto
  • text-align: center
  • (image) display: block, margin-right: auto, margin-left: auto, width: x%
  • position: relative, absolute; right: 0
  • float: right

Vertically align elements

  • padding: 25px, 0
  • flexbox, justify content, align-items
  • position: absolute; top: 50%

CSS combinators

To format descentant, child or sibling items, you can use this combinators:

  • div p
  • div > p
  • div + p
  • div ~ p
